Capacidades

Diseñado para Desarrolladores

Un ecosistema agéntico completo sin salir de tu terminal.

Motor de Parseo Universal

Otorga capacidades agénticas a cualquier LLM interpretando intenciones desde lenguaje natural, sin depender del Tool Calling nativo.

Equipo Multi-Agente

BashAgent orquesta, ResearcherAgent analiza (solo lectura) y CodeAgent edita con precisión quirúrgica. Tres especialistas, un objetivo.

Seguridad Human-in-the-Loop

Confirmación antes de comandos destructivos, vista previa de diffs, ediciones atómicas. El agente propone, tú apruebas.

Ubicuidad Agéntica

Arquitectura cliente-servidor: controla el mismo agente desde la TUI en tu PC y desde Telegram en tu móvil simultáneamente.

RAG Semántico

Indexa tu repositorio completo con embeddings vectoriales. Los agentes conocen toda tu base de código sin leer cada archivo.

Comandos Mágicos

%models, %reset, %undo, %compress. Control total del flujo agéntico con prefijos intuitivos.

Instalación

Listo en 30 Segundos

Un solo comando lo configura todo: entorno virtual, rutas y API keys.

bash 
curl -fsSL https://raw.githubusercontent.com/gatovillano/KogniTerm/main/install.sh | bash

El script instala el entorno virtual, configura tu PATH y te guía para añadir tus API keys.

💡 Si tu shell (ej. fish) o terminal da el error curl: (23), usa: curl -fsSL -O https://raw.githubusercontent.com/gatovillano/KogniTerm/main/install.sh && bash install.sh && rm install.sh

bash 
pipx install kogniterm

Instala en un entorno aislado. Recomendado si ya usas pipx.

bash 
pip install kogniterm

Instalación estándar con pip. Asegúrate de estar en un virtualenv.
